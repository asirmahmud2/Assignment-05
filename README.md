1. What is the difference between var, let, and const?
let and var allows you to reassign its value later, whereas const requires an initial value and cannot be reassigned.

2. What is the spread operator (...)?
The spread operator make array into individual value. 

3. What is the difference between map(), filter(), and forEach()?
map() make each item in an array and returns a completely new array
filter() returns a new array containing only the items that pass a specific condition.
forEach() executes a provided function once for each array element like loop but does not return a new array.

4. What is an arrow function?
An arrow function is a simple and small way of writing function expressions in JavaScript.

5. What are template literals?
Template literals enable easy string integration, allow embed variable.
